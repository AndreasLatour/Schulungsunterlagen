# Arbeitsverzeichnis
Bei der Installation wird eine SVWS-Arbeitsverzeichnis angelegt, in dem sämtliche Nutzerdaten gespeichert werden. Hier liegen unter anderem:
* Reports
* Datenbanksicherungen
* Connection-Files
* ggf. die Dokumentenverwaltung
* Exporte
